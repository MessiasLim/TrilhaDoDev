## Como usar esta trilha
 
### Para aprender
 
Siga a progressão recomendada abaixo. Cada etapa usa o que você aprendeu na anterior.

Use `Ctrl+F` dentro de qualquer arquivo para buscar um conceito específico.
 
### Para ensinar
 
Você pode reutilizar qualquer conteúdo desta trilha no seu material.

## Como executar os exemplos
 
### Opção 1 — Live Server (recomendado)
 
1. Abra a pasta no VS Code
2. Instale a extensão **Live Server**
3. Clique com o botão direito em qualquer `.html` → **Open with Live Server**
### Opção 2 — Node.js
 
```bash
npx http-server . -c-1 -p 8080
```
Acesse `http://localhost:8080` no navegador.
 
### Opção 3 — Python
 
```bash
python3 -m http.server 8080
```
Acesse `http://localhost:8080` no navegador.
