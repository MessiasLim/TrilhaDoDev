# Curso Básico de Frontend

Este repositório reúne **2 anos de estudos organizados de forma
didática** sobre:

-   HTML
-   CSS
-   JavaScript

É uma coleção de **lições, exemplos práticos e exercícios** voltados
para iniciantes que desejam construir uma base sólida em desenvolvimento
Frontend.


# Objetivo

Fornecer um material estruturado para:

-   Aprender HTML do básico ao semântico
-   Entender CSS (fundamentos, layout e responsividade)
-   Praticar JavaScript (sintaxe e DOM)
-   Resolver exercícios progressivos
-   Revisar conceitos de forma organizada


# Estrutura do Projeto

    .
    ├── html/         → Conceitos e exemplos de HTML
    ├── css/          → Fundamentos, layout, componentes e responsividade
    ├── javascript/   → Sintaxe e manipulação básica do DOM
    ├── exercicios/   → Enunciados e desafios práticos
    ├── markdow/      → Material sobre Markdown
    └── README.md

## CSS

Dividido em:

-   **fundamentos/** → seletores, box-model, display, posicionamento\
-   **layout/** → flexbox, grid, alinhamento\
-   **componentes/** → botões, cards\
-   **estilos/** → cores, fontes, variáveis\
-   **responsivo/** → media queries


## JavaScript

Separado em:

-   **sintaxe/**
    -   variáveis e tipos\
    -   operadores\
    -   condicionais\
    -   loops\
    -   arrays\
    -   objetos\
    -   funções
-   **dom/**
    -   manipulação básica de elementos (ainda será adiciona)

# Como Usar

## Clonar o projeto

```sh
git clone https://github.com/MessiasLim/TrilhaDoDev.git
```


## Rodar com Live Server (Recomendado)

Instale a extensão:

-   Live Server

Clique com botão direito em um arquivo `.html` → **Open with Live
Server**


## Usar servidor simples via Node

```sh
npx http-server . -c-1 -p 8080
```

Depois abra no navegador:

    http://localhost:8080
